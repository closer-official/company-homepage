# Cursor Prompt Templates

## 1) 新しい Cursor で最初に貼る
```text
このリポジトリのデザイン統一を最優先で作業してください。
作業前に docs/brand-style-guide.md と .cursor/rules/design-consistency.mdc を必ず読んでください。
既存トークン（色・余白・タイポ）を使い、新しい色や派手な演出を勝手に追加しないでください。
変更後は「統一感を守るために維持した点」を3つ報告してください。
```

## 2) 新規ページ追加
```text
新規ページを追加してください。既存サイトのトーンを厳守してください。
要件:
- max-w-5xl か max-w-2xl を使う
- headingはfont-heading、本文はtext-[var(--text-muted)]中心
- 主要CTAはterracotta系、二次導線はテキストリンク
- スマホで情報過密にしない
```

## 3) 既存ページ改善（デザイン崩さない）
```text
このページを改善してください。ただしブランドトーンは変えないでください。
禁止:
- 新色追加
- 過剰なアニメーション
- 厚いカードUIへの変更
必須:
- 既存のコピー温度感を維持
- 余白と可読性を上げる
```

## 4) テキストだけ調整
```text
文言だけ改善してください。レイアウト・色・クラス構成は極力変えないでください。
トーン:
- 丁寧
- 熱量はあるが押し売りしない
- 読み手に行動を促す
```

## 5) 最終チェック依頼
```text
この変更がブランド統一に沿っているかレビューしてください。
出力形式:
1. 良い点
2. 崩れている点
3. 最小修正案（3点）
```
