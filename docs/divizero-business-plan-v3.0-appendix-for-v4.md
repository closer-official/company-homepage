## 0. 法人化のタイミング・軍資金・撤退リスク（ver.3.0 新設）

**v1.0・v2.0**: 本章の見出し・本文なし（法人化タイミングは未記載）。

**v3.0（新設）**:

### 法人成りに採用しない基準

- 世間一般で言われる **節税目的の売上800万円** という基準は**採用しない**。

### 法人成りを実行する条件（両方を完全に満たすタイミングのみ）

1. **信用の獲得とスケールアップの必要性**  
   事業を2倍・3倍へ拡大するプロセスにおいて、**社会的な信用（法人格）**があることでより大きな契約や展開が見込めるフェーズに到達したか。
2. **収益モデルの確実な立証**  
   現在のビジネスモデルにおける**収益性が完全に確認**でき、規模を拡大すれば**それに比例して収益も確実に増加する**という見込みが立ったか。

### 軍資金の確保と撤退リスクの管理

- **初期段階から法人化して身動きが取りにくくなるリスク**を避ける。
- まずは**機動力のある現在の形態**でテストマーケティング的に動き、**どこかの分野で確実に事業を確立**させる。
- **うまくいかなかった場合の撤退のしやすさ**を維持しながら、**運営の軍資金を積み上げる**ことを最優先とする。

  - **数ヶ月の常設**

### 【収益構造（ユニット・エコノミクス）】（ver.1.0 全文維持）

- 総経費（場所代・車両等）を「100」とした場合、3分割の時間枠それぞれに「50」で転貸する。
- 3枠合計「150」を回収し、「50」を手残り（利益率33.3%）とする。
- 3ヶ月単位の契約とし、中途撤退時の返金は一切行わない。

**v1.0 → v2.0（モデル表現）**

- **v1.0**: 上記は **3分割の時間枠** を前提とした数式。
- **v2.0**: **実運用の枠数は 3 に固定しない**。収益の考え方（経費に対する転貸・手残り）は維持しつつ、**2 枠・4 枠等の設計**を §「時間枠・ゴールデンタイム」で補足（数式の「3枠合計150」は **3分割モデルとしての参照例** として残す）。

### 【学生報酬】（ver.1.0 全文維持）

- 場所を確定させ、オーナーを付けた学生に、手残りのうち10,000円を成功報酬として即時支給。

### ver.2.0 追記：保健所・責任の切り分け（v1.0 未確定変数4への回答）

**v1.0（未確定としての問い）**: 同一車両・場所での「時間帯別営業許可」の自治体交渉。

**v2.0（方針案）**:

- **食品提供に関する許認可などは各店主（個人）が取得**。
- **出店に関する手続き・車両・場所の枠組みなど、大きな箱**はディビゼロ側が責任を負う形を想定し、自治体・保健所との対話で責任所在を明確にする。

### ver.2.0 追記：時間枠・ゴールデンタイム・デッドタイムの価値（v1.0 未確定変数9への回答の一部）

**v1.0（未確定としての問い）**: アイドルタイムに5万円払うオーナーが実在し続けるか。

**v2.0（価値提供の考え方）**:

- **ゴールデンタイムの例**: モーニング、ランチ、ディナー、ナイト（居酒屋系）など。これらに応じて出店枠を設計。
- **枠数**: 3に固定しない。**4枠は集まりにくい懸念**。**2枠は「最初から」ではなく、3枠で足りない人向けの追加プラン**としても可。
- **デッドタイムの第二の価値（移動広告）**:
  - 例：ある時間帯の枠では、実売上が立ちやすいのは朝と昼のピークに偏るなどし、**空いた時間帯は渋谷等の一等地を指定ルートで走行**。
  - 車体に広告を載せ、「あの走ってるやつ」認知を取る。掲載により **広告収入を複数ソースから獲得**。
  - オーナー向け：**本来月8万円の枠が、走行条件付きで月5万円**など、時間帯に応じた **割引・パッケージ** で提供する（または最初からその価格帯のみ用意）。
  - 時間帯分割の例：6:00〜14:00 / 14:00〜22:00 / 22:00〜翌6:00 など。

### ver.2.0 追記：スケール・損益分岐の捉え方（v1.0 未確定変数10への回答の一部）

**v1.0（未確定としての問い）**: 提携何件で「加盟したほうが原価が下がる」状態に到達するか。

**v2.0（訴求の枠組み）**:

- **提携件数だけでは測れない価値**をオーナーに訴求する。
- **例示**: 利益率30％・初期費用200万の事業 vs 利益率15％・初期費用10万の事業を、実績ゼロで前者を選ぶのは非合理的。**後者でスモールスタートし固定ファンを作ってから拡大**する方が自然。
- **他社サービスとの差**: 期間限定イベント型の場所提供が多く、**毎回場所を探す手間**がボトルネックになりがち。**一等地での常設・回転の仕組み**でその手間を削ることが強み。

## 4. 兵隊（学生ギルド）の統治と増殖

### 【獲得フロー】（ver.1.0 全文維持）

- **ターゲット**: SNSのマイクロ層（10投稿以下）や、大学生活に不満を持つ層。
- **スタンス**: 「稼がせる」という甘い勧誘ではなく、「店舗DX支援の業務委託パートナー」としての事務的な公募。
- **初動**: 5名〜50名を確保。

### 【行動指針・歩留まり】（ver.1.0 全文維持）

- **鉄の規律**: 50件のアタック（DM・訪問）に対し、1件の有料成約を最低基準とする。
- **確率論**:
  - 導入率：1/10（50件中5件が無料導入）
  - 転換率：1/5（5件中1件が有料転換）
- **増殖**: 「友人紹介報酬3,000円」を営業利益から捻出する仕組みにより、紹介した側が教育も担う自律増殖型組織。

**v2.0 → v3.0（紹介報酬の拠出元の明確化）**

- **v2.0 まで**: v1.0 に同じく「**営業利益から捻出**」と記載（§4 増殖）。
- **v3.0**: 詳細は §2「友達紹介報酬の仕組み（ver.3.0 で精緻化）」のとおり。**運営取り分（10,800円）から**、被紹介者の**初回1件成約時のみ** 3,000円を拠出。2件目以降は運営に満額 10,800円。

### ver.2.0 追記：稼働イメージ・離脱方針（v1.0 未確定変数6への回答）

**v1.0（未確定としての問い）**: 1件の成約（50件アタック）までに学生が挫折しないためのケア。
