# Tadanosuke Closer Brand Style Guide

## 目的
- ページを追加・改修しても、サイト全体の雰囲気を崩さないための基準。
- 判断に迷ったら「高級感より親しみ」「派手さより温度感」「情報整理」を優先する。

## ブランドトーン
- キーワード: 編集的 / 余白 / 温かさ / 信頼感 / 速度感。
- 避ける: サイバー感、蛍光色、重い黒、過剰なガラス表現、装飾過多。
- コピーは断定しすぎず、誠実さと熱量を両立する。

## カラーパレット（固定）
- `--bg-base: #f2f0eb`
- `--bg-card: #ebe9e4`
- `--text: #3d3935`
- `--text-muted: #6b6560`
- `--terracotta: #b8956f`
- `--terracotta-soft: #d4c4b0`
- `--sage: #8b9a7a`
- `--sage-soft: #a8b498`
- `--border: #ddd9d2`

### ルール
- 新色を勝手に追加しない。必要なら既存変数で表現する。
- CTA は `--terracotta` 系を優先。補助導線は `--sage` か text link。
- 純黒 `#000` / 純白 `#fff` の広範囲使用は避ける。

## タイポグラフィ
- 見出し: `font-heading`（Noto Serif JP 系）
- 本文: `font-sans`（Noto Sans JP 系）
- 見出しは短く強く、本文は `leading-relaxed` か `leading-loose` を基本。

## レイアウト
- コンテナ幅: `max-w-5xl`（標準） / `max-w-2xl`（読み物）
- 上部余白: 基本 `pt-[72px]` + section `py-16 sm:py-24`
- 情報は「カード乱用」より「文字と余白」で見せる。

## コンポーネント規約
- ヘッダー文言・順序は維持:
  - Services / サービス一覧・仕事依頼
  - About / 想い
  - Company / 会社概要
  - Partners / パートナー募集
  - Contact / お問い合わせ
- ボタン:
  - 主要CTA: 丸ボタン + `bg-[var(--terracotta)]`
  - 二次CTA: テキストリンク or ボーダーボタン
- 法務リンク:
  - `Company` に `特定商取引法` / `プライバシーポリシー` を置く

## モーション
- 原則ゆっくり（0.8s〜1.2s）。
- `prefers-reduced-motion` に配慮する。
- 「目立つためのアニメーション」は禁止。意味のある導線補助のみ。

## 画像方針
- 背景は落ち着いた建築/ワークスペース系。彩度は低め。
- 人物は誠実さ優先。無理にストック人物を使わない。
- 推奨サイズ:
  - ヒーロー背景: 1920x1080 以上
  - About人物: 800x1000（3:4）
  - OG: 1200x630
- 著作権・ライセンス不明画像は使わない。

## 文言方針
- サイト中核コピー:
  - 「Webを、もっと自由な遊び場に。」
  - 「情報だけのサイトは、もういらない。」
- 文体は丁寧語。断定しすぎず、価値は具体で示す。

## 実装前チェック
- 既存CSS変数だけで表現できているか
- 余白が詰まりすぎていないか
- CTAの優先度が明確か
- スマホ表示で1画面情報量が過密でないか
